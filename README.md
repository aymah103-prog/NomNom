A recipe search and diet planning web application built using HTML, CSS, JavaScript, Python Flask, and SQLite.

## Features
- User signup & login  
- Database connectivity (SQLite)  
- Contact form saving data  
- Diet plan page  
- Admin notification panel  
- Responsive UI

## Tools Used
- Python 3  
- Flask  
- SQLite  
- HTML / CSS / JS

## How to run locally
1. Install Python  
2. Install Flask  
3. Run `app.py`  
4. Open: http://127.0.0.1:5000/
